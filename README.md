# LifeTrifling
Just record the life trifling.


## first case

```
已知：年化收益率为4%，每月定投5000，求：
1. 按每天结算，利滚利的方式，计算1年后的收益（假设一年360天, 每月按30天算，也即每30天增投5000）
2. 按每天结算，利滚利的方式，计算10年后的收益（假设一年360天）
3. 按每月结算，利滚利的方式，计算1年后的收益
4. 按每月结算，利滚利的方式，计算10年后的收益

解答：

1-2题，
	每天收益率：dR = (1 + 0.04) ^ (1 / 360) - 1 = 0.000108952360303

	1年后本金：5000 * 12 = 60000
	1年后本金+收益：5000 * (1 + dR) ^ 360 + 5000 * (1 + dR) ^ 330 + ... + 5000 * (1 + dR) ^ 30 = 61292.21053167043

	10年后本金：600000
	10年后本金+收益：735880.8454460986

3-4题，
	每月收益率：mR = (1 + 0.04)^(1 / 12) - 1 = 0.0032737397822

	1年后本金：5000 * 12 = 60000
	1年后本金+收益：5000 * (1 + mR) ^ 12 + 5000 * (1 + mR) ^ 11 + ... + 5000 * (1 + mR) ^ 1 = 61292.210531671095

	10年后本金：600000
	10年后本金+收益：735880.8454461766

原来不用如此复杂：
	每年收益率：yR = 0.04

	1年后本金：5000 * 12 = 60000
	1年后本金+收益：5000 * (1 + yR) ^ (11 / 12) + 5000 * (1 + yR) ^ (10 / 12) + ... + 5000 * (1 + yR) ^ (1 / 12) = 61292.210531671066

	10年后本金：600000
	10年后本金+收益：735880.8454461738
```
